## Quick Start

### Compile & Run
```bash
cd "Lab 1"
gcc 1.1-lab.c -o 1.1-lab
./1.1-lab

# In Windows
gcc 1.1-lab.c -o 1.1-lab.exe
./1.1-lab.exe
```

### For math functions, link the math library:
```bash
cd "Lab 1"
gcc 1.2-lab.c -lm -o 1.2-lab
./1.2-lab
```
